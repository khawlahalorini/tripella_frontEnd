# user Story 
* As traveler, I want an application that provides me the places available at my destination, so that I can choose the place that I will visit.
* As traveler, I want a page where I can review the available places, so I know where I can go.
* As traveler, I want a page where I can add new places, so that I can publish it.
* As traveler, I want a feature that enables me to add places to my favorites, so that I can save the places I like. 
*  As traveler, I want a feature that enables me to add lists of cities that I will visit, so that I can divide my trips. 
*  As traveler, I want a feature that enables me to add places to my list, so that I can display places that belong to each trip.
